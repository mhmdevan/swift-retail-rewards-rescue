# Legacy Folder

This folder hosts legacy adapters/modules that mirror existing production constraints.
- `LegacyNetworking/` Alamofire-first legacy client adapters (active in Phase 3)
- `LegacyUI/` storyboard/xib/view-controller assets retained during migration
- `LegacyStorageAdapters/` compatibility storage seams
- `LegacyLoyaltyKit/` Objective-C interoperability module (planned Phase 8)
